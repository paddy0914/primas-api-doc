# Primas Node API Documentation

## Group APIs

### Get group metadata

[GET] /groups/{group_dna}

### Create group

[POST] /groups

### Dismiss group

[DELETE] /groups/{group_dna}

### Get group members

[GET] /groups/{group_dna}/members

### Join group

[POST] /groups/{group_dna}/members

### Quit group

[DELETE] /groups/{group_dna}/members/{person_dna}

### Get group content

[GET] /groups/{group_dna}/content
